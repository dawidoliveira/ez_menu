## Descrição
EZ Menu é um app para restaurantes, onde o cliente consegue utilizar o app como cardápio.

## ✅ Plugins do Vscode

- Dart
- Dracula Oficial (tema, nao é obrigatorio esse, mas fica estiloso)
- Flutter
- Material Icon Theme
- Dart Data Class Generator
